## Commandes apprises

git remote add origin URL  → connecter repo local à GitHub
git remote -v              → vérifier la connexion distante
git push -u origin main    → premier envoi vers GitHub
git push                   → envoi normal après le premier
git clone URL              → copier un repo GitHub sur son PC
git pull                   → récupérer les changements distants

## Différence Git vs GitHub

Git    → outil installé sur mon PC
       → stocke les versions localement
       → fonctionne sans internet

GitHub → site web dans le cloud
       → accessible partout
       → visible par les recruteurs
