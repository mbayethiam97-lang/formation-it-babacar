# Lundi Sem.1 — Linux bases + VM Ubuntu

## Ce que j'ai fait
- Installé VirtualBox + VM Ubuntu 22.04 (4GB RAM / 50GB)
- Ouvert le terminal pour la première fois
- Créé le dossier /projets/reseau/semaine1

## Commandes apprises

pwd                → affiche le dossier où je suis actuellement
ls                 → liste les fichiers du dossier actuel
ls -la             → liste avec détails et fichiers cachés
cd /etc            → aller dans le dossier /etc
cd ..              → remonter d'un niveau vers le dossier parent
mkdir nom          → créer un nouveau dossier
touch fichier.txt  → créer un fichier vide

## L'arborescence Linux

/etc    → fichiers de configuration du système
/home   → dossiers personnels des utilisateurs  
/var    → logs et données variables
/tmp    → fichiers temporaires (vidés au redémarrage)

## Différence Linux vs Windows
- Tout part de / et non de C:\
- Linux est sensible à la casse : Fichier ≠ fichier
- Le terminal est plus puissant que l'interface graphique

## Ce que j'ai compris aujourd'hui

- Chaque commande fait une seule chose mais la fait bien
